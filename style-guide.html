Tekst & content

  • Gebruik je-vorm (geen u).
  • Schrijf in zin-hoofdletters: alleen de eerste letter van de zin met hoofdletter; geen woorden in ALL CAPS.
  • Koppen met sterke typografie: gewichten 800–900 voor hiërarchie.
  • Hover-states subtiel; focus-ring in merk-kleur voor toegankelijkheid.
  • Subtiele, moderne brand-gradients zijn toegestaan (paginatitel, menubalk, footer). Houd het rustig en zonder flashy effecten. Donkere modus wordt niet ondersteund.
  • Beeldgebruik: plaats afbeeldingen bij voorkeur naast tekst; houd ze subtiel en niet te dominant.

Underline en hover intent

Topmenu-links hebben een subtiele underline-animatie en kleurwisseling naar merk-kleur.

<a class="mainnav-link hover:text-brand" href="#">Categorie</a>
Mega-menu gedrag

Paneel opent onder nav, met lichte elevatie. Hover/focus op links geeft achtergrondtint en underline. aria-expanded op de trigger.

<button class="mainnav-link" aria-expanded="false">Hygiëne</button>
<div class="mega-panel" data-state="open">...</div>
Mobiele drawer

Offcanvas gebruikt fade voor overlay en slide voor panel. Sluiten via overlay, close-knop of ESC.

<div id="offcanvas" class="hidden">
  <button data-close="offcanvas" class="hdr-overlay fixed inset-0 bg-black/40"></button>
  <aside class="hdr-drawer fixed inset-y-0 left-0 w-80 bg-white">...</aside>
</div>
Sticky-on-scroll

Header verbergt/tonen met soepele translatie. Respecteert reduced motion.

header[role="banner"].is-hidden { transform: translateY(-100%); }
A11y
  • Gebruik aria-expanded op triggers; focus-ring zichtbaar.
  • Focus-trap in modals; sluit met ESC.
  • Navigatielinks en mega-links moeten :focus-visible outline krijgen in merk-kleur.

Bestellijst tabel

  • Kolommen: Positie, Artikelnr, Productnaam, Minimale voorraad, Maximale voorraad, Huidige voorraad, Besteladvies, Prijs, Acties.
  • Sticky header op desktop. Mobiel stapelt tot kaarten met dezelfde velden.
  • Inline validatie: min ≤ max, huidig ≥ 0; focus-ring in merk-kleur.
  • Besteladvies = max(0, max - huidig); bij 0 een zachte "OK"-badge.
// JS
function computeAdvice({min,max,current}){ return Math.max(0, max - current); }
